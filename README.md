# 体重管理アプリ — PWA

## ファイル構成
```
weight-app/
├── index.html      ← メインアプリ
├── manifest.json   ← PWA設定
├── sw.js           ← Service Worker（オフライン対応）
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

---

## 無料で公開する方法（おすすめ: Netlify）

### 1. Netlify（一番かんたん）
1. https://netlify.com にアクセス → 無料登録
2. ダッシュボードに `weight-app` フォルダをそのままドラッグ＆ドロップ
3. 自動で `https://xxxxx.netlify.app` というURLが発行される
4. 完成！

---

### 2. GitHub Pages（GitHubを使っている場合）
1. GitHubで新しいリポジトリを作成
2. ファイルをアップロード
3. Settings → Pages → Branch: main で有効化
4. `https://ユーザー名.github.io/リポジトリ名` でアクセス可能

---

### 3. Vercel
1. https://vercel.com に登録
2. `weight-app` フォルダをアップロード
3. 自動デプロイ完了

---

## スマホへのインストール方法

### Android（Chrome）
1. アプリをブラウザで開く
2. 右上メニュー →「ホーム画面に追加」
3. または画面下部に表示される「追加する」バナーをタップ

### iPhone（Safari）
1. Safariでアプリを開く
2. 下部の共有ボタン（四角から矢印が出るアイコン）をタップ
3. 「ホーム画面に追加」を選択

---

## 機能
- 社員ごとの体重記録・グラフ表示
- 目標体重の設定
- 管理者による全社員一覧確認
- オフライン動作対応
- データはブラウザに保存（端末ごとに管理）

---

## 注意事項
- データはブラウザのlocalStorageに保存されます
- 同じ端末・同じブラウザでのみデータが共有されます
- 複数の端末でデータを共有したい場合は、バックエンド（データベース）の追加が必要です
